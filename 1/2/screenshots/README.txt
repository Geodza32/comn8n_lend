Примеры скриншотов лендинга

Здесь будут находиться скриншоты различных секций лендинга.

Чтобы увидеть реальный вид сайта, требуется запустить проект:
1. Установить Node.js
2. Выполнить команды:
   - npm install
   - npm run dev
3. Открыть http://localhost:3000 в браузере

Основные секции лендинга:
- Hero секция с заголовком "Сообщество, где автоматизация превращается в прибыль"
- Секция "Что даёт тебе сообщество"
- Секция "Как ты получаешь пользу с первых дней"
- Секция "Ты перестаёшь быть исполнителем"
- Секция "Что ты найдёшь внутри"
- Секция "Подписка"
- Секция "Кто создал сообщество"
